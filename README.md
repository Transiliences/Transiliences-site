# Site Transiliences

Projet Next.js prêt pour déploiement sur Vercel.

## Déploiement rapide

1. Dézipper le dossier.
2. Envoyer tous les fichiers dans le dépôt GitHub `transiliences-site`.
3. Retourner sur Vercel.
4. Importer le dépôt.
5. Cliquer sur Deploy.

Le site utilise Next.js, Tailwind CSS, Framer Motion et Lucide React.
